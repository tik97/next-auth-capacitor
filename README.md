## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Run on IOS

```bash
npx cap add ios
npm run static
npx cap sync
npx cap run ios
```

Run on android

```bash
npx cap add android
npm run static
npx cap sync
npx cap run android
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
